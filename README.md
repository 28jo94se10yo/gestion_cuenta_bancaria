# Sistema de Gestión Bancaria (POO)

Este proyecto es un simulador de backend bancario desarrollado en **Python** utilizando el paradigma de **Programación Orientada a Objetos (POO)**. Está diseñado para gestionar de manera eficiente múltiples cuentas bancarias de forma simultánea.

### 🚀 Funcionalidades Principales:
* **Administración de Cuentas:** Creación y gestión de perfiles financieros con datos protegidos.
* **Operaciones de Caja:** Implementación lógica para depósitos y extracciones con validación de saldo disponible.
* **Transferencias:** Capacidad de realizar movimientos de fondos entre diferentes instancias de objetos (cuentas).
* **Escalabilidad:** Estructura preparada para manejar un volumen de hasta 50 cuentas o más mediante colecciones de datos.

### 🛠️ Conceptos Técnicos Aplicados:
* **Clases y Objetos:** Modelado de entidades financieras (Cuenta, Cliente, Banco).
* **Encapsulamiento:** Protección de los atributos sensibles (como el saldo) mediante métodos de acceso seguros.
* **Lógica de Negocio:** Validación estricta de transacciones para evitar saldos negativos o errores de tipo de dato.

### 💻 Cómo ejecutarlo:
1. Asegúrate de tener Python instalado.
2. Descarga el archivo del proyecto (ej: `gestor_bancario.py`).
3. Ejecútalo desde tu terminal o IDE con el comando: `python gestor_bancario.py`
